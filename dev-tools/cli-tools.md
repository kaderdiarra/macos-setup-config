# CLI Tools

Useful command-line utilities to install via Homebrew.

## Install All

```bash
brew install wget telnet tldr
```

## Tools

| Tool | Description |
|------|-------------|
| `wget` | Download files from the web via command line (not built into macOS) |
| `telnet` | Network debugging tool (not built into macOS) |
| `tldr` | Simplified man pages — shows common usage examples for CLI commands |
