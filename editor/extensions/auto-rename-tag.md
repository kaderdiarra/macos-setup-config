# Auto Rename Tag

Automatically renames the paired HTML/XML tag when you edit one.

- **Extension ID**: `formulahendry.auto-rename-tag`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag

## Install

**GUI**: `Cmd+Shift+X` → search "Auto Rename Tag" → Install

**CLI**:
```bash
code --install-extension formulahendry.auto-rename-tag
```
