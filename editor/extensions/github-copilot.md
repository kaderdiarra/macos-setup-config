# GitHub Copilot

AI-powered code completion — suggests lines, functions, and blocks of code in real time as you type.

- **Extension ID**: `GitHub.copilot`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=GitHub.copilot
- **Pricing**: Free tier (limited completions/month), Pro $10/mo, free for students (GitHub Education)

## Install

**GUI**: `Cmd+Shift+X` → search "GitHub Copilot" → Install

**CLI**:
```bash
code --install-extension GitHub.copilot
```

## Setup

1. After install, sign in with your **GitHub account** when prompted
2. Authorize the Copilot extension via GitHub OAuth
