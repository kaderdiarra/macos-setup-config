# Font Size Shortcuts

Change editor font size with keyboard shortcuts without zooming the entire VS Code UI. Useful for presentations.

- **Extension ID**: `fosshaas.fontsize-shortcuts`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=fosshaas.fontsize-shortcuts

## Install

**CLI**:
```bash
code --install-extension fosshaas.fontsize-shortcuts
```
