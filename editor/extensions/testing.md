# Testing Extensions

Test runner integrations for Jest and Playwright.

## Jest Runner

Run/debug individual Jest tests from context menu and CodeLens.

- **Extension ID**: `firsttris.vscode-jest-runner`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=firsttris.vscode-jest-runner

## Jest (by Orta)

Runs Jest tests automatically in the background with inline results and coverage.

- **Extension ID**: `orta.vscode-jest`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=orta.vscode-jest

## Playwright

Integrates Playwright test runner — run, debug, and view traces directly in VS Code.

- **Extension ID**: `ms-playwright.playwright`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=ms-playwright.playwright

## Install All

```bash
code --install-extension firsttris.vscode-jest-runner
code --install-extension orta.vscode-jest
code --install-extension ms-playwright.playwright
```
