# Todo Tree

Shows all TODO, FIXME, HACK, and other comment tags in a tree view — quickly browse and jump to every tagged comment across your entire project.

- **Extension ID**: `Gruntfuggly.todo-tree`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree

## Install

```bash
code --install-extension Gruntfuggly.todo-tree
```

## Usage

- Open the **Todo Tree** panel in the sidebar (tree icon in activity bar)
- Click any item to jump to that location
- Default tags: `TODO`, `FIXME`, `BUG`, `HACK`
- Customizable tags and colors in settings
