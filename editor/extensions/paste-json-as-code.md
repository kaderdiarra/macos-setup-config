# Paste JSON as Code

Convert JSON from clipboard into typed code (TypeScript, Python, Go, C#, Java, etc.). Use `Cmd+Shift+P` → "Paste JSON as Code".

- **Extension ID**: `quicktype.quicktype`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=quicktype.quicktype

## Install

**CLI**:
```bash
code --install-extension quicktype.quicktype
```
