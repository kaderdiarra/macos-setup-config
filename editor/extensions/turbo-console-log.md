# Turbo Console Log

Insert meaningful `console.log` statements instantly with keyboard shortcuts — automatically includes the variable name, file name, and line number.

- **Extension ID**: `ChakrounAnas.turbo-console-log`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=ChakrounAnas.turbo-console-log

## Install

```bash
code --install-extension ChakrounAnas.turbo-console-log
```

## Shortcuts

- `Ctrl+Alt+L` — Insert `console.log` for selected variable
- `Alt+Shift+D` — Delete all inserted console.log statements
- `Alt+Shift+C` — Comment out all inserted console.log statements
- `Alt+Shift+U` — Uncomment all inserted console.log statements
