# Todo Highlight

Highlights `TODO`, `FIXME`, and other annotation keywords in your code so they stand out and don't get missed.

- **Extension ID**: `wayou.vscode-todo-highlight`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=wayou.vscode-todo-highlight

## Install

**GUI**: `Cmd+Shift+X` → search "Todo Highlight" → Install

**CLI**:
```bash
code --install-extension wayou.vscode-todo-highlight
```
