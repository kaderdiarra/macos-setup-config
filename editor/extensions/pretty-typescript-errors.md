# Pretty TypeScript Errors

Makes TypeScript error messages human-readable — formats them with syntax highlighting and better structure instead of a blob of text.

- **Extension ID**: `yoavbls.pretty-ts-errors`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=yoavbls.pretty-ts-errors

## Install

**CLI**:
```bash
code --install-extension yoavbls.pretty-ts-errors
```
