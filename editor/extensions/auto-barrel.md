# Auto Barrel

Automatically generates and updates barrel (`index.ts`) export files for your project.

- **Extension ID**: `imgildev.vscode-auto-barrel`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=imgildev.vscode-auto-barrel

## Install

**CLI**:
```bash
code --install-extension imgildev.vscode-auto-barrel
```
