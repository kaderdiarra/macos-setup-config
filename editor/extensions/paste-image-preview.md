# Paste Image & Image Preview

Two companion extensions for working with images directly in your editor.

## Paste Image

Paste images from clipboard directly into Markdown or code files — auto-saves the image to your project and inserts the reference.

- **Extension ID**: `mushan.vscode-paste-image`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=mushan.vscode-paste-image

## Image Preview

Shows image previews in the editor gutter and on hover for image paths/URLs in your code.

- **Extension ID**: `kisstkondansen.vscode-gutter-preview`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=kisstkondansen.vscode-gutter-preview

## Install

**GUI**: `Cmd+Shift+X` > search each extension ID > Install

**CLI**:
```bash
cursor --install-extension mushan.vscode-paste-image
cursor --install-extension kisstkondansen.vscode-gutter-preview
```
