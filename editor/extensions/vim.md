# Vim Extension

Full Vim emulation (modal editing, keybindings, Ex commands) inside Cursor / VS Code.

- **Extension ID**: `vscodevim.vim`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=vscodevim.vim
- **Repo**: https://github.com/VSCodeVim/Vim

## Install

**GUI**: `Cmd+Shift+X` > search `vscodevim.vim` > Install

**CLI**:
```bash
cursor --install-extension vscodevim.vim
```

## Learning Vim

### Vim Cheatsheet

Quick-reference Vim keybinding cheatsheet inside the editor.

- **Extension ID**: `andenetalexander.vim-cheatsheet`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=andenetalexander.vim-cheatsheet

### Learn Vim

Interactive tutorial that teaches Vim motions and commands within VS Code.

- **Extension ID**: `vintharas.learn-vim`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=vintharas.learn-vim

```bash
code --install-extension andenetalexander.vim-cheatsheet
code --install-extension vintharas.learn-vim
```
