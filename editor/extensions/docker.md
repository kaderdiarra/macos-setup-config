# Docker Extension

Manage containers, images, and Docker Compose directly from VS Code — includes Dockerfile IntelliSense and one-click debugging.

- **Extension ID**: `ms-azuretools.vscode-docker`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-docker

## Install

**GUI**: `Cmd+Shift+X` → search "Docker" → Install (by Microsoft)

**CLI**:
```bash
code --install-extension ms-azuretools.vscode-docker
```
