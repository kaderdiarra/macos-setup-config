# Project Manager

Switch between projects quickly — save, organize, and open any project with a single click or shortcut.

- **Extension ID**: `alefragnani.project-manager`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=alefragnani.project-manager

## Install

```bash
code --install-extension alefragnani.project-manager
```

## Usage

- `Cmd+Shift+P` → **Project Manager: Save Project** — save current workspace
- `Cmd+Shift+P` → **Project Manager: List Projects** — switch to a saved project
- `Alt+Cmd+P` — Quick open project list
- Projects appear in the sidebar under the Project Manager panel
