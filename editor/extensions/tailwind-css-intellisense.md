# Tailwind CSS IntelliSense

Autocomplete, syntax highlighting, and linting for Tailwind CSS classes — hover over classes to preview the real CSS.

- **Extension ID**: `bradlc.vscode-tailwindcss`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss

## Install

**GUI**: `Cmd+Shift+X` → search "Tailwind CSS IntelliSense" → Install

**CLI**:
```bash
code --install-extension bradlc.vscode-tailwindcss
```
