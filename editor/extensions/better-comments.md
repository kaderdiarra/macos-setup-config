# Better Comments

Color-codes comments by type — alerts (`!`), queries (`?`), TODOs, highlights, and strikethroughs for improved readability.

- **Extension ID**: `aaron-bond.better-comments`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments

## Install

**CLI**:
```bash
code --install-extension aaron-bond.better-comments
```
