# CodeSnap

Take beautiful screenshots of your code with your current VS Code theme and font settings. Select code, run `Cmd+Shift+P` → "CodeSnap".

- **Extension ID**: `adpyke.codesnap`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=adpyke.codesnap

## Install

**CLI**:
```bash
code --install-extension adpyke.codesnap
```
