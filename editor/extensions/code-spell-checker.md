# Code Spell Checker

Catches common spelling mistakes in code and comments — essential when pushing code to production.

- **Extension ID**: `streetsidesoftware.code-spell-checker`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker

## Install

**GUI**: `Cmd+Shift+X` → search "Code Spell Checker" → Install

**CLI**:
```bash
code --install-extension streetsidesoftware.code-spell-checker
```
