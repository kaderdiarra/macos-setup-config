# Auto Close Tag

Automatically inserts the closing tag when you type an opening HTML/XML tag.

- **Extension ID**: `formulahendry.auto-close-tag`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-close-tag

## Install

**CLI**:
```bash
code --install-extension formulahendry.auto-close-tag
```
