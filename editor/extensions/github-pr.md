# GitHub Pull Requests & Issues

Review and manage GitHub pull requests and issues directly from VS Code — browse, checkout, comment, and merge without leaving the editor.

- **Extension ID**: `GitHub.vscode-pull-request-github`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=GitHub.vscode-pull-request-github

## Install

```bash
code --install-extension GitHub.vscode-pull-request-github
```

## Setup

1. Sign in to GitHub when prompted after install
2. Open the **GitHub** panel in the sidebar to see PRs and issues
3. Click a PR to view diff, leave comments, approve, or merge
