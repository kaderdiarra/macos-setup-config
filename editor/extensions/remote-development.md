# Remote Development

Develop on remote machines, containers, or WSL as if local.

## Remote - SSH

Open folders on any remote machine over SSH.

- **Extension ID**: `ms-vscode-remote.remote-ssh`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-ssh

## Dev Containers

Develop inside Docker containers with full VS Code support.

- **Extension ID**: `ms-vscode-remote.remote-containers`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers

## Remote Explorer

Browse and manage remote connections.

- **Extension ID**: `ms-vscode.remote-explorer`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=ms-vscode.remote-explorer

## Install All

```bash
code --install-extension ms-vscode-remote.remote-ssh
code --install-extension ms-vscode-remote.remote-containers
code --install-extension ms-vscode.remote-explorer
```
