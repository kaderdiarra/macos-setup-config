# Language Support Extensions

Additional language support beyond JavaScript/TypeScript.

## YAML

YAML validation, completion, and schema integration.

- **Extension ID**: `redhat.vscode-yaml`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=redhat.vscode-yaml

## MDX

MDX language support — syntax highlighting and IntelliSense for JSX in Markdown.

- **Extension ID**: `unifiedjs.vscode-mdx`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx

## Install All

```bash
code --install-extension redhat.vscode-yaml
code --install-extension unifiedjs.vscode-mdx
```
