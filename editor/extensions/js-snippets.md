- **Extension ID**: `xabikos.javascriptsnippets`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=xabikos.javascriptsnippets

## Install

**CLI**:
```bash
code --install-extension xabikos.javascriptsnippets
```
