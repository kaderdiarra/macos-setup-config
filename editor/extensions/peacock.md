# Peacock

Color-code your VS Code workspaces — assign a unique color to the title bar, status bar, and activity bar per project so you can instantly tell which workspace you're in.

- **Extension ID**: `johnpapa.vscode-peacock`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=johnpapa.vscode-peacock

## Install

```bash
code --install-extension johnpapa.vscode-peacock
```

## Usage

- `Cmd+Shift+P` → **Peacock: Change to a Favorite Color**
- Pick from presets (Angular Red, Vue Green, React Blue, etc.) or set a custom hex
- Colors are saved per workspace in `.vscode/settings.json`
