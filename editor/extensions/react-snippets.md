# React Snippets

Two complementary snippet extensions for React development.

## Simple React Snippets

Curated set of React snippets — components, hooks, lifecycle methods.

- **Extension ID**: `burkeholland.simple-react-snippets`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=burkeholland.simple-react-snippets

## ES7+ React/Redux/React-Native Snippets

Full ES7+ snippets for React, Redux, and React Native with customizable shortcuts.

- **Extension ID**: `dsznajder.es7-react-js-snippets`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets

## Install All

```bash
code --install-extension burkeholland.simple-react-snippets
code --install-extension dsznajder.es7-react-js-snippets
```
