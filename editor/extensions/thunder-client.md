# Thunder Client

Lightweight REST API client inside VS Code — test APIs directly from the editor sidebar. Clean alternative to Postman.

- **Extension ID**: `rangav.vscode-thunder-client`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=rangav.vscode-thunder-client

## Install

**GUI**: `Cmd+Shift+X` → search "Thunder Client" → Install

**CLI**:
```bash
code --install-extension rangav.vscode-thunder-client
```
