# PostCSS Intellisense and Highlighting

Syntax highlighting and CSS intellisense for PostCSS files and modern CSS syntax. Works better than other similar extensions.

- **Extension ID**: `vunguyentuan.vscode-postcss`
- **Marketplace**: https://marketplace.visualstudio.com/items?itemName=vunguyentuan.vscode-postcss

## Install

**CLI**:
```bash
code --install-extension vunguyentuan.vscode-postcss
```
