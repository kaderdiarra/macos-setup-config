# Keka

macOS file archiver — compress and extract 7z, ZIP, TAR, GZIP, RAR, ISO, and more. Replaces the built-in Archive Utility.

- **Website**: https://www.keka.io

## Install

```bash
brew install --cask keka
```

## Setup

1. Open Keka and set as default extractor: `Keka > Settings > Set Keka as default extractor`
2. Drag to Dock for quick drag-and-drop compression
