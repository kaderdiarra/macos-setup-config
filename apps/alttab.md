# AltTab

Windows-style alt-tab for macOS — shows full window previews and lets you switch between individual windows, not just apps.

- **Website**: https://alt-tab-macos.netlify.app
- **Repo**: https://github.com/lwouis/alt-tab-macos

## Install

```bash
brew install --cask alt-tab
```

## Setup

1. Grant **Accessibility** permission in `System Settings > Privacy & Security`
2. Grant **Screen Recording** permission for window thumbnail previews
3. Configure key bindings and appearance in Preferences (menubar icon > Preferences)
4. Optionally replace `Cmd+Tab`: in AltTab settings → Controls → change shortcut to `Cmd+Tab`
