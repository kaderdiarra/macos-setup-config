# Stats

Free, open-source system monitor for the menu bar — CPU, GPU, memory, disk, network, battery, and clock with multiple time zones.

- **Repo**: https://github.com/exelban/stats

## Install

```bash
brew install --cask stats
```

## Setup

1. Launch Stats — modules appear as icons in the menu bar
2. Click each module's icon to configure which metrics are shown
3. Go to Settings to enable/disable modules, choose widget styles (mini, bar, graph), and set update intervals
4. **Battery**: enable percentage display, then merge widgets to combine icon + percentage
5. **Clock**: enable 24-hour format, add extra time zones (click clock to see them all)
6. Use with **Hidden Bar** to keep menu bar clean — drag Stats icons to preferred visibility
