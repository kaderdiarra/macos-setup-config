# Raycast

Spotlight replacement for macOS. Launch apps, manage windows, run scripts, clipboard history, and integrate with dev tools — all from a single keyboard shortcut.

- **Website**: https://www.raycast.com
- **Download**: https://www.raycast.com/download

## Install

```bash
brew install --cask raycast
```

## Setup

1. Launch Raycast and grant **Accessibility** and **Automation** permissions in `System Settings > Privacy & Security`
2. Set hotkey — default is `Option + Space`. To replace Spotlight with `Cmd + Space`:
   - Disable Spotlight shortcut: `System Settings > Keyboard > Keyboard Shortcuts > Spotlight`
   - Set Raycast hotkey to `Cmd + Space`
3. Browse the built-in **Raycast Store** to install extensions (GitHub, Spotify, Cursor, etc.)
4. Optionally sign in with a Raycast account to sync settings across devices

## Recommended Extensions

Install via Raycast: `Cmd+Space` → Settings (`Cmd+,`) → Extensions → Install from Store

- **Brew** — search and install Homebrew packages directly from Raycast
- **Tailwind CSS** — search utility classes and docs
- **Curl** — quick API route testing
- **Iconify** — search icon packs and copy SVG code
- **Color Picker** — pick colors anywhere on screen
