# Rectangle

Free, open-source window management for macOS — snap and resize windows with keyboard shortcuts.

- **Website**: https://rectangleapp.com

## Install

```bash
brew install --cask rectangle
```

## Setup

1. Launch Rectangle and grant **Accessibility** permissions in `System Settings > Privacy & Security`
2. Enable **Launch at Login** in preferences

## Default Shortcuts

| Action | Shortcut |
|--------|----------|
| Left Half | `Ctrl+Option+←` |
| Right Half | `Ctrl+Option+→` |
| Top Half | `Ctrl+Option+↑` |
| Bottom Half | `Ctrl+Option+↓` |
| Maximize | `Ctrl+Option+Return` |
