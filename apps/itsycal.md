# itsycal

Tiny menu bar calendar — shows the current date and a monthly calendar view with events from your macOS Calendar app.

- **Website**: https://www.mowglii.com/itsycal/
- **Repo**: https://github.com/sfsam/Itsycal

## Install

```bash
brew install --cask itsycal
```

## Setup

1. Launch itsycal and grant **Calendar** access when prompted
2. Enable **Launch at Login** in Preferences
3. Under Appearance, choose outline style, show month and day of the week
4. Customize the date/time format pattern if needed (e.g., `E MMM d`)
