# Arc

Clean, Chromium-based browser with sidebar tabs and workspaces. All Chrome DevTools and extensions work out of the box.

- **Website**: https://arc.net

> **Note**: Active development has stopped, but the browser is considered feature-complete.

## Install

```bash
brew install --cask arc
```

## Setup

1. Sign in with an account (required for sync)
2. Import bookmarks/passwords from Chrome/Safari/Firefox on first launch
3. **Create Spaces** — color-coded tab groups (e.g., Work, Personal) — switch with `Cmd+Option+Arrow`
4. Pin important tabs at the top of the sidebar (unpinned tabs auto-archive after 12h)
5. Set up **Profiles** under `Settings > Profiles` to isolate cookies/logins per Space

## Key Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+S` | Toggle sidebar |
| `Cmd+T` | New tab |
| `Cmd+Shift+C` | Copy current URL |
| `Cmd+Option+N` | Little Arc window (mini popup) |
