# iTerm2

Feature-rich terminal emulator for macOS — split panes, search, autocomplete, and extensive customization. Replaces the default Terminal app.

- **Website**: https://iterm2.com

## Install

```bash
brew install --cask iterm2
```

## Setup

1. Make it default: `iTerm2 > Make iTerm2 Default Term`
2. Install **Shell Integration**: `iTerm2 > Install Shell Integration` — enables command history, directory tracking
3. Enable **Natural Text Editing**: `Preferences > Profiles > Keys > Key Mappings > Presets > Natural Text Editing` — makes `Option+Arrow`, `Cmd+Delete` work as expected
4. Set font: `Preferences > Profiles > Text` — use a Nerd Font if using Oh My Zsh themes
5. Pick a color preset: `Preferences > Profiles > Colors > Color Presets`
6. Set appearance to **Minimal**: `Preferences > Appearance > Theme > Minimal` — cleaner title bar
7. **Reuse previous session directory**: `Preferences > Profiles > General > Working Directory` → "Reuse previous session's directory"
