# DBeaver

Universal database management tool — supports PostgreSQL, MySQL, SQLite, MongoDB, Oracle, SQL Server, and more. Visual SQL editor, ER diagrams, and data browsing.

- **Website**: https://dbeaver.io

## Install

```bash
brew install --cask dbeaver-community
```

## Setup

1. Launch DBeaver (skip the sample database prompt)
2. `Database > New Database Connection` → pick your DB type → enter host/port/credentials
3. Click **Test Connection** — the required JDBC driver downloads automatically on first connect
