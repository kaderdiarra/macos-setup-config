# Time Out

Break timer app — reminds you to take regular breaks to prevent repetitive strain injuries. Configurable normal breaks and micro breaks.

- **Website**: https://www.dejal.com/timeout/

## Install

```bash
brew install --cask time-out
```

## Setup

1. Configure two break types:
   - **Normal break**: 10 min every hour
   - **Micro break**: 10-20 seconds every 10 minutes
2. Enable **Launch at Login**
3. Grant permissions so it can detect if you're actively using the computer
4. Free version covers core functionality
