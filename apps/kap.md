# Kap

Open-source screen recorder — export to GIF, MP4, WebM, and APNG. Lightweight and clean.

- **Website**: https://getkap.co
- **Repo**: https://github.com/wulkano/Kap

## Install

```bash
brew install --cask kap
```

## Setup

1. Grant **Screen Recording** and **Accessibility** permissions when prompted
2. Click the menu bar icon to start recording (selected area or full screen)
