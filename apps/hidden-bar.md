# Hidden Bar

Free menu bar manager — hide icons from the top bar to keep it clean. Free alternative to Bartender.

- **Repo**: https://github.com/dwarvesf/hidden

## Install

```bash
brew install --cask hiddenbar
```

## Setup

1. Launch Hidden Bar and grant **Accessibility** permissions in `System Settings > Privacy & Security`
2. A `|` separator appears in the menu bar
3. `Cmd + drag` any menu bar icon to the left of the separator to hide it
4. Click the `>` arrow to toggle hidden icons visibility
5. Enable **Start at Login** in the app's preferences
