# NotchNook

Turns the MacBook notch into a functional utility area — media controls, file tray, notes, calendar, and more.

- **Website**: https://lo.cafe/notchnook
- **Pricing**: $25 one-time purchase (free trial available)
- **Requires**: MacBook with notch (2021 Pro+ or M2+ Air)

## Install

```bash
brew install --cask notchnook
```

## Setup

1. Grant **Accessibility** and **Screen Recording** permissions when prompted
2. Configure widgets/shelves via the app's preferences
3. Enable **Launch at Login**

## Usage

- Hover the notch to access media controls, notes, and file tray
- **File tray**: drag files to the notch area, then drag them to a destination — no need for extra Finder windows
- **Gestures**: swipe left/right with two fingers on the notch to skip songs
